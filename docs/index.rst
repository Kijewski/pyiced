PyIced
======

Installation
------------

.. code:: bash

    $ pip install pyiced

To install from source you need recent version of `Rust <https://www.rust-lang.org/>`_ installed in
your <tt>$PATH</tt>. See e.g. `rustup <https://rustup.rs/>`_.
